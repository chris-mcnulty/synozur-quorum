# Icons

This folder contains the Quorum brand icons required by the Microsoft 365
Agents Toolkit:

| File | Size | Description |
| --- | --- | --- |
| `color.png` | 192×192 px | Full-colour icon — dark navy (#1E1B2E) with gold Q ring. |
| `outline.png` | 32×32 px | White Q outline on a transparent background. |

Both icons are generated by the project and match the Quorum brand palette.
If you need to replace them, ensure the dimensions and format stay the same;
Microsoft will reject the sideload package otherwise.
